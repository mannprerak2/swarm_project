Genetic algorithm based scheduler for Engifest.

## Setup
1. Install [dart](https://dart.dev/)
2. Run`dart pub get`

## Running

#### Verbose output
```bash
dart bin/schedule.dart -v
```

#### Minimal output (once every 100 generation).
```bash
dart bin/schedule.dart
```

#### Output to file
```bash
dart bin/schedule.dart > output.txt
```
